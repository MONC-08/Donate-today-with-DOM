What is DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of nodes, where each node represents part of the document, such as elements, attributes, or text.

What is the purpose of the DOM?

The DOM allows developers to interact with and manipulate HTML and XML documents programmatically using languages like JavaScript. You can use the DOM to:

  • Modify content
  • Change styles
  • Add or remove elements
  • Handle events
How can you access elements in the DOM?

Using JavaScript, elements can be accessed in various ways:

  • document.getElementById('id'): Selects an element by its ID.
  • document.getElementsByClassName('class'): Selects elements by their class name.
  • document.getElementsByTagName('tag'): Selects elements by their tag name.
  • document.querySelector('selector'): Selects the first element that matches a CSS selector.
  • document.querySelectorAll('selector'): Selects all elements that match a CSS selector.
